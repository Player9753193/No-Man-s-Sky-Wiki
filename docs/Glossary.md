# 术语表（Glossary）

- **单位 Units**：通用货币
- **纳米星团 Nanites**：升级/科技相关货币
- **快速银 Quicksilver**：特殊活动货币
- **曲速 Warp**：星系间跃迁
- **曲速电池 Warp Cell**：曲速燃料（不同版本/翻译可能有别名）
- **反物质 Antimatter**：曲速燃料链关键材料
- **精炼 Refining**：通过精炼机进行配方转换
- **热点 Hotspot**：基地可利用的气体/矿物/电磁热点
- **哨兵 Sentinels**：星球/设施防卫单位
- **海盗 Pirates**：太空/星系冲突相关敌对单位

> 规则：同义词统一到一个主条目页面；本表只作为“命名裁决”。
