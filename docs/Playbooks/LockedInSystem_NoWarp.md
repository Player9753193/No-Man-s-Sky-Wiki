# Playbook：跳不动（No Warp）应急手册

## 目标
在曲速燃料不足时，快速恢复“可跃迁能力”，并避免再次卡死。

## 症状
- 曲速引擎无法充能 / 缺少关键材料
- 在高冲突/无法无天系统被骚扰，补给困难

## 最短解（优先级从上到下）
1) **空间站终端/商人直接买**（最快，但依赖库存）
2) **在本系统内收集关键材料**（稳定，但需要跑图）
3) **建立最小基地补给点**（一劳永逸的方向）

## 预防（工程化思路）
- 永远保持：曲速燃料 ≥ N 次跃迁的安全库存
- 把氧气/碳系资源做成稳定供给（见资源条目）
- 进入“无法无天”前先补满燃料与护盾消耗品
