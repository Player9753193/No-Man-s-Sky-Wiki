# 精炼（Refining）

## 概览
精炼系统负责把“原料”转换成“更高价值/更高密度/更可用”的材料，是工业化的核心。

## 你现在最相关的意义
- 把采集压力从“无限手采”转为“固定配方+稳定产出”
- 为曲速燃料链、基地供给链提供基础能力
