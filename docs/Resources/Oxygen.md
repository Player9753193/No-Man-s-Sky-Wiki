# 氧气（Oxygen）

| 字段 | 值 |
|---|---|
| 类型 | 资源（Resource） |
| 核心用途 | 生命维持、配方链、扩增类精炼 |
| 常见获取 | 终端购买 / 星球采集 / 气体热点工业化 |
| 参见 | [精炼系统](../Systems/Refining.md) |

## 简介
氧气是中期开始最关键的“工业底料”之一：既解决生存消耗，也常用于精炼与量产链。

## 用途
- 生命维持补给
- 精炼/合成链的原料（与碳系资源常耦合）
- 工业化基地的稳定供给资源

## 获取方式
### 早期（应急）
- 空间站/贸易站终端购买（受星系与库存影响）
- 星球地表植物/矿点采集（效率低）

### 中期（稳定）
- 固定路线补货：在资源更友好的星系循环购买

### 后期（工业化）
- 建立 **气体热点** 基地：提取器 + 储存罐 + 供电，实现稳定产出（建议作为第一批工业基地）

## 常见坑
- 只靠手采会把游戏体验拖进“永远缺氧”的循环
- 终端库存不稳定：工业化是根解
